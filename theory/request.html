# 简介

这是一个请求工具包，其中封装了

# request

# _validateLoginState()

作用：校验登陆态

通过缓存类中get方法获取缓存中的登陆态，检验登陆态是否过期，和refreshKey是否过期判断是否需要重新登陆

# _refreshLoginState(refresh_key)

作用：发送请求更新登陆秘钥

# _req(__options)

作用：发出请求

参数：请求相关的数据

如果需要登陆，则将获取本地缓存中的登陆态中的skey，将其赋给请求头部的skey属性

发送请求后检验登陆态是否有效，如果失效了则刷新后重新登陆，并且重新回调_req()发送请求；如果刷新后重新登陆失败，则注销并且抛出异常。

最后返回请求响应的内容

# 请求部分

封装了请求默认值和请求附带的额外信息

# Raven.captureBreadcrumb

作用：sentry埋点

wx.request不可扩展，因此只能手动收集请求的行为：例如在经过封装的请求函数的成功回调内添加：

Raven.captureBreadcrumb({
  category: 'ajax',
  data: {
    method: 'get',
    url: 'weapp.showcase.page/1.0.0/get',
    status_code: 200
  }
