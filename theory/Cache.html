# 简介

这个是一个缓存类，封装了小程序storage接口

# get(symbolKey)

作用：获取指定的缓存

方法：通过this[key]先判断内存中是否存在，存在则返回key；内存中不存在则从存储中读取wx.getStorageSync(key)

# set(symbolKey, value)

作用：异步写入存储

方法：wx.setStorage({ key, data: value })

# remove(symbolKey)

作用：清除指定缓存

方法：wx.removeStorage({ key })
