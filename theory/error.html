# 简介

这是一个统一的错误提示工具

# ErrorMessage类

# 作用

将传入的信息抛出并上传错误信息到sentry

# 参数

  • msg：错误信息
  • code：错误码
  • options：可选项
属性 类型 说明
needReport Boolean 判断是否将错误信息上报sentry系统

在构造函数里先声明一个Error()实例对象error，将options赋给变量option

然后判断错误码是否存在typeof code === 'string'，存在的话就将error的信息加上错误码，不存在则将code赋给option，这里是为了防止传入的第二个参数options

# showError(options)

# 作用

通用的错误弹窗

# 参数

options是传入的信息

Object.assign() 方法用于将所有可枚举属性的值从一个或多个源对象复制到目标对象，并返回目标对象。

通过用Object.assign()将弹框信息都复制到optionObj里，最后调用wxShowModal显示弹窗
